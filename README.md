### `> ./mission.sh`

```txt
Most compliance work breaks down because it lives in PDFs,
spreadsheets, screenshots, and forgotten tickets.

I am interested in the opposite:

- live risk signals
- cleaner evidence
- faster reviews
- better workflows
- security practices teams can actually use

The goal is simple:
make governance more measurable, more useful, and less painful.
```

### `> ls ./projects --impact`

| Project | What it does | Stack |
|---------|--------------|-------|
| [**VenderScope**](https://github.com/darkyzowo/venderscope) | A continuous vendor risk intelligence platform built to replace static spreadsheets with live monitoring. Tracks breach exposure, CVEs, Shodan-visible assets, weighted risk scoring, and ISO-style reporting. | JavaScript |
| [**ContraAI**](https://github.com/darkyzowo/contraai) | An AI-assisted contract review platform that turns legal playbooks into structured clause analysis, helping teams review contracts faster and spot risky or missing terms. | Next.js · Claude API |
| [**ZARAK_OS**](https://github.com/darkyzowo/zarak-os) | A cyber-noir portfolio OS with a custom desktop, terminal emulator, and immersive interface, built to feel more like a system than a static personal site. | TypeScript · Three.js |
